Diagrammes de décision binaires en Java.

Ce projet vise à fournir une implémentation efficace et facile à utiliser de diagrammes de décision binaires en Java. Il peut être utilisé pour résoudre des problèmes complexes en utilisant une approche basée sur des décisions binaires.

Installation:

1-Téléchargez le code source du projet depuis GitHub.
2-Ouvrez le projet dans votre IDE Java préféré (Eclipse ou IntelliJ).
3-Importer les bibliothèques nécessaires (par exemple, JUnit pour les tests).
4-Compilez et exécutez le projet.



Utilisation:


dans ce programme on offre la possibilité de tester la BDD en introduisant une formule booléenne dans le fichier main,
et le programme vous déssinera l'arbre complet associé à la formule booléenne ainsi que l'arbre associé à la réduction.
